# eslint-to-editorconfig

Module for transforming configuration from ESLint to EditorConfig format.

## Install

```
npm install --global eslint-to-editorconfig
```

## Usage

Run it in the same directory as your .eslintrc

```
eslint-to-editorconfig
```
